Help: X-Transam

Description

X-Transam è un modulo per tradure da una lingua ad un'altra lingua.

Install/uninstall

Non sono necessarie misure particolari, seguire il processo di installazione standard - estrarre la cartella / xtransam nella cartella ../modules. Installare il modulo tramite Amministrazione -> Modulo di sistema -> Moduli. Se avete bisogno di istruzioni dettagliate su come installare un modulo, vedere il Manuale XOOPS Operations.

Help: X-Transam

Description

X-Transam is a module to translate from one language to another language.

Install/uninstall

No special measures necessary, follow the standard installation process - extract the /xtransam folder into the ../modules directory. Install the module through Admin -> System Module -> Modules. If you need detailed instructions on how to install a module, please see the XOOPS Operations Manual.
